# JSON API 1.2 Changelog

## 17-03-2022
### Добавлено
- Добавлен отчёт [Текущих Остатков](https://dev.moysklad.ru/doc/api/remap/1.2/reports/#otchety-otchet-ostatki-tekuschie-ostatki).

### Изменено
