## Выгрузка статистики запросов к API

В МоемСкладе доступен отчет **Статистика использования API**. Он позволяет выгрузить запросы к API от аккаунта совершенные за последние 24 часа, что может быть полезно для:
* идентификации всех учетных записей, которые используют API - такая проверка перед деактивацией учетных записей или сбросом паролей позволяет избежать остановки в работе связанных с ними интеграций;
* оценить наиболее активное время использования API - количество запросов к JSON API в единицу времени ограничено (подробнее см. в [разделе](#/restrictions#3-limity-i-ogranicheniya-v-json-api)), поэтому интенсивные синхронизации данных лучше распределить по времени, чтобы ограничения не мешали работе. Например, если на аккаунте подключена интеграция или приложение, которое синхронизирует товары, заказы или другие документы каждую нулевую минуту часа, то другие интеграции стоит запускать после пиковой нагрузки.
* локализовать причины ошибок при асинхронной работе. Например, при превышении лимита параллельных запросов или параллельной работе с одной и той же сущностью (удаление/изменение). 
* отслеживать статистику использования API

В отчете указываются следующие данные:

* время начала обработки;
* пользователь;
* метод;
* путь;
* время обработки (в миллисекундах);
* код ответа;
* код remap (заполняется при получении ошибки на запрос).

### Как посмотреть отчет Статистика использования API

* Перейдите в раздел **Настройки → Обмен данными → Экспорт**.

![useful image](./images/workbook/export_api.png?raw=true)

* Нажмите на кнопку **Экспортировать** и выберите **Статистика запросов к API**. Отчет загружается на компьютер в виде файла в формате CSV.

![useful image](./images/workbook/report_api.png?raw=true)
